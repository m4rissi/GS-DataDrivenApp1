# Global Solution - Data Driven Application

Projeto desenvolvido em Python para monitoramento ambiental utilizando dados de satélites.

## Funcionalidades

- Cadastro de eventos ambientais
- Cálculo de área afetada
- Média de intensidade
- Região com maior número de ocorrências
- Relatório final dos eventos

## Tecnologias utilizadas

- Python
- Listas
- Estruturas de repetição
- Estruturas condicionais

## Como executar

1. Abra o arquivo .py
2. Execute o código no VS Code
3. Digite os dados pedidos pelo sistema
4. Veja o relatório final
